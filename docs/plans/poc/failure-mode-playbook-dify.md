## 条目模板

### FM-NN: <一句话症状>

- **症状(证据)**:Pod 状态 / 日志关键行 / 事件。
- **根因**:为什么会这样(结合 README/镜像文档)。
- **RuntimeState 分类**:对应 troubleshooter 决策树的哪一态。
- **修法**:用哪个 MCP 工具、改了什么值。
- **验证**:重部署后如何确认已解决。
- **是否需文档输入**:纯日志能不能定位?还是必须查官方文档?
- **人工确认点**:是否触发了半自动确认。
